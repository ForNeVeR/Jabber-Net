Jabber-Net

JID

Namespace: JabberNet.jabber


Provides simple JID management.

Constructors

ConstructorDescription
new(jid)
Signature: jid:string -> unit

Creates a JID from a string. This will parse and perform the stringprep (RFC 3454) process.

new(user, server, resource)
Signature: (user:string * server:string * resource:string) -> unit

Builds a new JID from the given components. This will parse and perform the stringprep (RFC 3454) process.

Instance memberDescription
Bare
Signature: string

Gets the username and XMPP server domain values of the JID. For example: user@example.com

BareJID
Signature: JID

Gets the user@server JID associated with this JID, as a JID. Slightly faster than building it yourself, since stringprep is avoided.
