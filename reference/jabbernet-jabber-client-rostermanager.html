Informs the client that a subscription request was received that cannot be auto-handled.

OnUnsubscription
Signature: UnsubscriptionHandler

Informs the client that an Unsubscribe/Unsubscribed notification from another user. By default, the user will be removed from the roster after this event fires. To prevent this, you need to set the remove property to false.
