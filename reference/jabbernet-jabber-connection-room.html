Jabber-Net

Room

Namespace: JabberNet.jabber.connection


Manages a multi-user conference room. See XEP-0045 (http://www.xmpp.org/extensions/xep-0045.html).

Instance members

Instance memberDescription
Ban(jid, reason)
Signature: (jid:JID * reason:string) -> unit

Ban a user from re-joining the room. Must be an admin.

ChangeAffiliation(...)
Signature: (jid:JID * affiliation:RoomAffiliation * reason:string) -> unit

Change the affiliation (long-term) with the room of a user, based on their real JID.

ChangeRole(nick, role, reason)
Signature: (nick:string * role:RoomRole * reason:string) -> unit

Change the role of a user in the room, by nickname. Must be a moderator.

Configure()
Signature: unit -> unit

Configures the room. OnRoomConfig MUST be set first. OnRoomConfig will be called back in the GUI thread if there is an InvokeControl on your XmppStream. Make sure that OnRoomConfig does not return until it has the answer, typically by popping up a modal dialog with the x:data form.

DefaultConfig()
Signature: unit -> unit

Determines whether to use the default conference room configuration or to retrieve the configuration form from the XMPP server.

GrantMembership(jid, reason)
Signature: (jid:JID * reason:string) -> unit

Make this user a member of the room.

GrantVoice(nick, reason)
Signature: (nick:string * reason:string) -> unit

Un-mute a muted user. Give them "voice".

Invite(invitee, reason)
Signature: (invitee:JID * reason:string) -> unit

Invite a user to join the room.

IsParticipating
Signature: bool

Have we joined the room successfully?

JID

The bare JID of the room. room@service

Signature: unit -> unit

Joins the room. If the room is created, Configure() will be called automatically.

Join(password)
Signature: password:string -> unit

Join a room, using a password.

Kick(nick, reason)
Signature: (nick:string * reason:string) -> unit

Leave(reason)
Signature: reason:string -> unit

Exits the room. This cleans up the entry in the ConferenceManager, as well.

MakeModerator(nick)
Signature: nick:string -> unit

Make this user a moderator of the room.

Signature: (parties:ParticipantCollection * reason:string * callback:IqCB * state:obj) -> unit

ModifyRoles(...)
Signature: (parties:ParticipantCollection * reason:string * callback:IqCB * state:obj) -> unit

Modify the roles of the parties in this list. To use, retrive a ParticipantCollection, change the roles of the parties in that collection, then pass that modified collection in here.

Nickname()
Signature: unit -> unit

The nickname that others in the room will see for you. Set has the side-effect of changing the nickname on the server.

OnJoin

OnLeave

OnParticipantPresenceChange

OnPresenceChange

OnPresenceError
Signature: RoomPresenceHandler

Informs the client that an error in response to a room join, nick change, or presence update has occurred.

A side-chat message.

OnRoomConfig
Signature: ConfigureRoom

Informs the client that the room configuration form was received. It is up to the listener to call the FinishConfig() method. The IQ in the callback is the parent of the x:data element.
