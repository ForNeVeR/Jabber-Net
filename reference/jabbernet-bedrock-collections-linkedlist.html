Jabber-Net

LinkedList

Namespace: JabberNet.bedrock.collections


A doubly-linked list implementation, with a sentinal wrap-around m_header. Yes, it seem like this should have been included in System.Collections. This may be a nicer implementation of Queue than the one in System.Collections, which uses an array. YMMV.
