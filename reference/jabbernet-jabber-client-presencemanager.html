GetFeatureJID(jid, featureURI)
Signature: (jid:JID * featureURI:string) -> JID

Get the most available full JID that implements the given feature. Unlike most routines in PresenceManager, may also return JIDs that have negative presence. If a full JID is specified, this is effectively the same as HasFeature, but null will be returned if the feature isn't implemented.

Returns: null if none found
GetFeatures(jid)
Signature: jid:JID -> StringSet

Get the features associated with the JID. If a bare JID is passed in, this will be a union of all of the features for all of the resources of this user. Otherwise, it will be the features for the given resource. Requires a CapsManager to be set before use.

Returns: Null if no features are known.
HasFeature(jid, featureURI)
Signature: (jid:JID * featureURI:string) -> bool

Does the given JID implement the given feature? Bare JID asks if any resource of that user implements that feature. Full JID asks if the given resource implements that feature. Requires a CapsManager to be set before use.

Returns: True if the feaure is implemented
IsAvailable(jid)
Signature: jid:JID -> bool

Determines if a specified JID is online with any resources. This performs better than retrieving the particular associated presence packet.

Returns: If true, the user is online; otherwise the user is offline
