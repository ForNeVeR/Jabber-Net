Instance members

Instance memberDescription
BeginFindServiceWithFeature(...)
Signature: (featureURI:string * handler:DiscoNodeHandler * state:obj) -> unit

Finds a component that implements a given feature, which is a child of the root. This will call back on the first match. It will call back with null if none are found.

BeginGetFeatures(node, handler, state)
Signature: (node:DiscoNode * handler:DiscoNodeHandler * state:obj) -> unit

Retrieves the features associated with this node and then calls back on the handler. If the information is in the cache, handler gets called right now.

BeginGetFeatures(...)
Signature: (jid:JID * node:string * handler:DiscoNodeHandler * state:obj * cache:bool) -> unit

Retrieves the features associated with this node and then calls back on the handler. If caching is specified, items already in the cache call the handler immediately.

BeginGetFeatures(...)
Signature: (jid:JID * node:string * handler:DiscoNodeHandler * state:obj) -> unit

Retrieves the features associated with this JID and node and then calls back on the handler. If the information is in the cache, handler gets called right now.

BeginGetItems(node, handler, state)
Signature: (node:DiscoNode * handler:DiscoNodeHandler * state:obj) -> unit

Retrieves the child items associated with this node, and then calls back on the handler. If the information is in the cache, handler gets called right now.

BeginGetItems(...)
Signature: (jid:JID * node:string * handler:DiscoNodeHandler * state:obj * cache:bool) -> unit

Retrieves the child items associated with this node, and then calls back on the handler. If caching is specified, items already in the cache call the handler immediately.

BeginGetItems(jid, node, handler, state)
Signature: (jid:JID * node:string * handler:DiscoNodeHandler * state:obj) -> unit

Retrieves the child items associated with this node and JID, and then calls back on the handler. If the information is in the cache, handler gets called right now.

GetNode(jid, node)
Signature: (jid:JID * node:string) -> DiscoNode

GetNode(jid)
Signature: jid:JID -> DiscoNode

Creates nodes where only the JID is specified.

Returns: If DiscoNode exists, returns the found node. Otherwise it creates the node and return it.

Root
Signature: DiscoNode

Gets the root node. This is probably the server that the client is connected to. If the Children property of this root node is null, the disco#items request has not returned an answer. Register on this node's OnFeatures callback.
